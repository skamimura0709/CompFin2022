print関数¶

数値や文字列を（画面に）出力する

print(1)

1

print(10)

10

計算式は評価されて表示される

print(1 + 2)

3

print(2 * 3)

6

文字列として表示したいときは' （シングルクォーテーション）または"（ダブルクォーテーション）でくくる

print('a')

a

print('Hellow World!')

Hellow World!

mathモジュール（数学関数）¶

数学関数は、モジュールとして提供される。例えば平方根の計算をしたい場合は import mathというおまじないを実行してから、math.sqrtという関数を使う。一般にmath.関数名という形で関数を使う。mathモジュールの詳細は https://docs.python.org/ja/3/library/math.html などを参照せよ。

半径\(r\)の円の面積は\(\pi r^2\)で計算できた。Pythonで半径\(r\)を入力して、円の面積を出力するプログラムは以下のようになる。円周率はmath.piで計算できる。

import math
r = 2
print(math.pi * r ** 2)

12.566370614359172

\(x\)についての2次方程式 \(ax^2+bx +c=0\)の解は

\[ x = \frac{-b - \sqrt{b^2-4ac}}{2a}, \frac{-b + \sqrt{b^2-4ac}}{2a} \]

により計算できる。\(a, b, c\)を入力し2次方程式の解を出力するプログラムを書け。平方根はmath.sqrtで計算できる。

# import math
a = 1
b = 2
c = -3
print((-b - math.sqrt(b**2 - 4 * a * c))/(2 * a))
print((-b + math.sqrt(b**2 - 4 * a * c))/(2 * a))

-3.0
1.0

Black-Scholes式¶

ヨーロピアンコールオプションの価格\(C\)は

\[\begin{split} C = S N(d_1) - K e^{-rT} N(d_2)\\ d_1 = \frac{\ln(S/K) + (r -\sigma^2/2)T}{\sigma \sqrt{T}}\\ d_2 = d_1 - \sigma \sqrt{T} \end{split}\]

によって計算できる。 ここで、\(S\)は現在の原資産価格、\(K\)は権利行使価格、\(T\)は満期までの時間、\(r\)は無リスク金利、\(\sigma\)は原資産のボラティリティ、\(N()\)は標準正規分布の分布関数を表す。

標準正規分布の分布関数は from scipy.stats import normとしてから、norm.cdfを使えばよい。

\(S = 100, K = 99, T = 1, r = 0.01, \sigma = 0.2\) のとき \(C\) は約 \(8.91\) となる。

from scipy.stats import norm
import math
S = 100
K = 99
T = 1
r = 0.01
sigma = 0.2
d1 = (math.log(S / K) + (r + 0.5 * sigma**2) * T)/(sigma * math.sqrt(T))
d2 = d1 - sigma * math.sqrt(T)
C = S * norm.cdf(d1) - K * math.exp(-r * T) * norm.cdf(d2)
print(C)

8.918504421014205
